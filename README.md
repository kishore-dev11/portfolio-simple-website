# Portfolio Website

This is my personal portfolio website created using **HTML** and **basic CSS**. The website is designed to introduce myself and provide a simple way for visitors to contact me.

The portfolio currently contains two main pages:

* **Home Page** – A simple introduction about me and my portfolio.
* **Contact Page** – A page where visitors can find my contact information.

## Technologies Used

* HTML
* CSS

## Purpose

This project was built as part of my learning journey in web development. It helped me understand the basics of structuring web pages with HTML and styling them using CSS.
